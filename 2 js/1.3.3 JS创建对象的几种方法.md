## JS创建对象的几种方法

#### 1. 构造函数创建对象

```js
function person(name, sex, age) {
	this.name = name;
    this.sex = sex;
    this.age = age;
    this.show = function() {
        console.log(this.name+"_"+this.sex+"_"+this.age);
    }
}
var obj1 = new person("zhangsan", "男", 18);
obj1.show();
var obj2 = new person("lisi", "女", 20);
```
