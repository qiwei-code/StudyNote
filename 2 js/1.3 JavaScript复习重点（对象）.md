## JavaScript复习重点（对象）









**Object对象本身的方法与Object实例方法**

* Object“本身的方法”就是直接定义在`Object`对象的方法，如：

```js
Object.print = function (o) { console.log(o) };
```

* Object实例方法就是定义在`Object`原型对象`Object.prototype`上的方法，可以被`Object`实例直接使用，如：

```js
Object.prototype.print = function () {
  console.log(this);
};
var obj = new Object();
obj.print() // Object
```
