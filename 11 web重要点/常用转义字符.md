## 常用转义字符

| 字符 | 转义字符 |   表示   |
| :--: | :------: | :------: |
|  n   |    \n    |  换行符  |
|  r   |    \r    |  回车符  |
|  t   |    \t    | 横向跳格 |
|  u   |    \u    | 编码转换 |
|  /   |    \/    |   斜杠   |
|  b   |    \b    |  退格符  |
|  f   |    \f    |  换页符  |
|  “   |    \“    |  双引号  |
|  ‘   |    \‘    |  单引号  |
|  \   |    \\    |  反斜杠  |
